# Business Insights Report - EDA

## Objective
Perform Exploratory Data Analysis on a digital marketing campaign dataset to identify high-performing channels and recommend budget allocation strategies.

## Dataset Features
- Customer demographics
- Campaign information
- Engagement metrics
- Conversion data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Referral campaigns achieved the highest conversion rate.
- Customers aged 35–45 converted most frequently.
- Loyalty points and previous purchases strongly influenced conversions.

## Recommendation
Increase budget allocation for Referral, PPC, and SEO campaigns while optimizing Social Media campaigns.
